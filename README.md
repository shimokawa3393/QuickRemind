## QuickRemind

**「思いついたら、すぐ登録。」**  
シンプルで分かりやすい、カテゴリー管理型のリマインダーアプリです。

---

### 特徴

- 即登録：思いついた予定をすぐに追加  
- カテゴリー管理：仕事・プライベート・支払いなどで整理  
- 通知リマインド：忘れたくない予定をプッシュ通知でお知らせ  
- シンプルUI：誰でも迷わず使える画面設計  
- カスタマイズ性：カテゴリの追加・編集も可能

---

### 使用技術

| 項目         | 使用技術               |
|--------------|------------------------|
| 開発環境     | Xcode + Swift          |
| 対応OS       | iOS 16.0以降           |
| 言語         | Swift, SwiftUI         |
| データ保存   | UserDefaults           |
| 通知機能     | UNUserNotificationCenter |
